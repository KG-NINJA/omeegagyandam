# Gyandam Lock-On Frame

## 概要
`index.html` 単体で動作する3Dロボットシューティング。照準中央で敵を捉えると最大5機までロックし、誘導レーザーの一斉射で撃破することができます。連続撃破や被弾でパワーが蓄積し、満タン時には一定時間ロック不要で撃ち放題となる「SUPER LOCKON」が発動します。

## ゲームプレイの流れ
1. ワープ演出後に交戦フェーズへ突入します。
2. 画面中央のロックリング内に敵を収めるとホールドゲージが進み、最大5機まで同時ロック。
3. スペースキーで通常ツインビーム、Enter/Xでロック済み目標へ誘導レーザーを放ちます。
4. 撃破や被弾でPOWERゲージが上昇し、100%でSUPER LOCKON。ロック無しで近傍目標を自動捕捉し続けます。
5. 旗艦コアを破壊するとステージがループし、スコアアタックを継続できます。

## 主な特徴
- **ロックオンHUD**: クロスヘア周囲に動的リングとターゲット番号を表示。視覚的にロック状態を把握可能。
- **誘導レーザー**: ロック数に応じて最大5連射。スーパー状態では金色残光の高威力モードに変化。
- **パワー/コンボシステム**: 連続撃破でコンボ倍率が上がりPOWERゲージが急増。被弾でも少量チャージされ、攻守の緊張感を演出。
- **多層艦隊AI**: ドレッドノートや戦艦、護衛機が各自のパターンで行動し、旗艦からは随時戦闘機が発艦。
- **通信/AIパネル**: HUD下部の通信ログと左上パネルで状況を表示し、没入感を強化。

## 操作方法
- `Arrow Keys` / `WASD`: ロール連動のヨー・ピッチ操作
- `Shift` / `Ctrl`: 加速・減速
- `Q` / `E`: 強制ロール
- `Space`: 通常ツインビーム
- `Enter` または `X`: 誘導レーザー（ロックしていれば最大5連、SUPER中は自動補足）
- `R`: 撃墜後リスタート
- `F1`: デバッグモード（無敵）切替

## 実行方法
ブラウザで `index.html` を直接開くだけで動作します。ローカルWebサーバを経由しても問題ありません。
